# Dirty Laundry (scaffold)

This workspace contains a minimal Next.js frontend and a Sanity Studio backend.

Quick start:

1. Install dependencies:
   ```powershell
   npm install
   cd studio
   npm install
   cd ..
   ```
2. Run Next.js dev server:
   ```powershell
   npm run dev
   ```
3. Run Sanity Studio (from root):
   ```powershell
   npm run studio
   ```

Environment:
- Edit `.env.local` and set `SANITY_API_TOKEN`
